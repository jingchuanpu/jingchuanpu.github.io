Three words are traced: revolutionary appears in Docs 1 & 2 only  ·  AI appears in Docs 1 & 2 only  ·  precision appears in Doc 3 only. Watch how TF-IDF rewards distinctiveness.

TF(t, d) = count of t in d ÷ total words in d

Measures how frequently a term appears within one document.

TF alone cannot distinguish "revolutionary" from "AI" — both score identically in Docs 1 and 2. We need IDF to separate them by corpus-wide rarity.

IDF(t) = log( N / df(t) )

N = total documents (3)  ·  df(t) = number of documents containing t  ·  log = natural log

WordAppears indf(t)CalculationIDF score
revolutionary Docs 1, 2 2 log(3/2) 0.405
AI Docs 1, 2 2 log(3/2) 0.405
precision Doc 3 only 1 log(3/1) 1.099
"precision" gets a much higher IDF (1.099 vs 0.405) because it is unique to one document — it is a more distinctive signal. A word in every document would score IDF = log(3/3) = 0 and be completely suppressed.

TF-IDF(t, d) = TF(t, d) × IDF(t)

High score = frequent in this document AND rare across the corpus. That word is the document's fingerprint.

The three document vectors are now numerically distinct — Docs 1 & 2 share a hype-weighted fingerprint while Doc 3 has a completely different, substantive-weighted profile. A cosine similarity between Doc 1 and Doc 2 would be high; between Doc 1 and Doc 3 it would be near zero. This is TF-IDF doing its job.
