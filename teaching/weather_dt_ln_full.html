Root entropy (using ln)

14 instances: 9 Yes, 5 No

H(root) = −[ (9/14)·ln(9/14) + (5/14)·ln(5/14) ]
         = −[ 0.6429·ln(0.6429) + 0.3571·ln(0.3571) ]
         = −[ 0.6429·(−0.4418) + 0.3571·(−1.0296) ]
         = −[ −0.2840 + (−0.3677) ]
         = −[ −0.6517 ]
         = 0.6517 nats

This is our baseline. We want to find which feature reduces this the most.

Using ln gives entropy in nats instead of bits (log₂). The conversion factor is ln(2) ≈ 0.6931, so 0.940 bits × 0.6931 = 0.6517 nats — same information, different units.
